## 信号语义（PLC / UI 必读）

| 旧 `result_code` | 新 `signal` | 说明 |
|------------------|-------------|------|
| 0 OK | -1 | 安全 / 区内无目标 |
| 1 SLOW | 1 | SLOW 确认 |
| 2 STOP | 2 | STOP 确认 |
| — | 0 | 有入侵未确认（过渡） |
| 3 故障 | -1 + FAULT 展示 | 勿与 SAFE 视觉混淆 |

禁止旧数值直写 PLC；须经 **SignalAdapter**（#20）；UI 接线见 #23。
